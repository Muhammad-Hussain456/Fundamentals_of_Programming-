### 2. Demonstrate Implicit and Explicit Type Conversions

* **Problem Definition**: Show how implicit and explicit type conversions work in C++.
* **Problem Analysis**: Implicit conversion happens automatically, whereas explicit conversion requires the use of casting operators.
* **Algorithm**:

  1. Demonstrate implicit conversion by assigning a `float` to an `int`.
  2. Demonstrate explicit conversion by casting an `int` to a `float`.
* **Program**:

  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
      // Implicit Conversion
      int intVar = 5;
      float floatVar = intVar;  // Implicit conversion from int to float
      cout << "Implicit conversion (int to float): " << floatVar << endl;
      
      // Explicit Conversion
      float num = 3.5;
      int result = (int) num;  // Explicit conversion from float to int
      cout << "Explicit conversion (float to int): " << result << endl;

      return 0;
  }
  ```

### 3. Program to Compute the Simple Interest Given Principal, Rate, and Time

* **Problem Definition**: Calculate the simple interest based on the formula: `SI = (P * R * T) / 100`.
* **Problem Analysis**: Accept `P` (Principal), `R` (Rate), and `T` (Time) as input and compute the interest.
* **Algorithm**:

  1. Input principal, rate, and time.
  2. Apply the formula to calculate the simple interest.
  3. Output the result.
* **Program**:

  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
      float principal, rate, time, interest;
      cout << "Enter principal amount: ";
      cin >> principal;
      cout << "Enter rate of interest: ";
      cin >> rate;
      cout << "Enter time in years: ";
      cin >> time;

      interest = (principal * rate * time) / 100;
      cout << "Simple Interest: " << interest << endl;

      return 0;
  }
  ```

### 4. Calculate Compound Interest Using Formula

* **Problem Definition**: Calculate the compound interest using the formula: `A = P * (1 + R/100)^T`, where `A` is the amount and `CI = A - P`.
* **Problem Analysis**: Accept the principal, rate, and time to compute compound interest.
* **Algorithm**:

  1. Input principal, rate, and time.
  2. Calculate the amount `A` using the formula.
  3. Calculate the compound interest by subtracting the principal from the amount.
  4. Output the compound interest.
* **Program**:

  ```cpp
  #include <iostream>
  #include <cmath>
  using namespace std;

  int main() {
      float principal, rate, time, amount, interest;
      cout << "Enter principal amount: ";
      cin >> principal;
      cout << "Enter rate of interest: ";
      cin >> rate;
      cout << "Enter time in years: ";
      cin >> time;

      amount = principal * pow((1 + rate / 100), time);
      interest = amount - principal;

      cout << "Compound Interest: " << interest << endl;

      return 0;
  }
  ```

### 13. Write a Program to Compute Volume of a Cylinder

* **Problem Definition**: Calculate the volume of a cylinder.
* **Problem Analysis**: Use the formula: `Volume = π * radius^2 * height`.
* **Algorithm**:

  1. Input radius and height.
  2. Apply the formula to calculate volume.
  3. Output the volume.
* **Program**:

  ```cpp
  #include <iostream>
  #define PI 3.14159
  using namespace std;

  int main() {
      float radius, height, volume;
      cout << "Enter radius of cylinder: ";
      cin >> radius;
      cout << "Enter height of cylinder: ";
      cin >> height;

      volume = PI * radius * radius * height;
      cout << "Volume of the cylinder: " << volume << endl;

      return 0;
  }
  ```
