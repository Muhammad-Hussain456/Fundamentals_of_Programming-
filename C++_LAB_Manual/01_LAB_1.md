## LAB 1: Introduction to Programming and C++ Basics

### Objectives:

* **Understand problem-solving using algorithms and flowcharts**
  Learn the basics of problem-solving using logical steps (algorithms) and visual representations (flowcharts).

* **Learn C++ program structure, syntax, and environment setup**
  Get familiar with the structure of a C++ program, understand basic syntax, and set up the development environment for C++ programming.

* **Compile and execute basic programs**
  Compile and run simple C++ programs to understand how the C++ code is translated into machine code and executed.

### 10. Write a Program to Convert Kilometers to Miles

* **Problem Definition**: Convert kilometers to miles.
* **Problem Analysis**: Use the conversion formula: `1 kilometer = 0.621371 miles`.
* **Algorithm**:

  1. Input the distance in kilometers.
  2. Multiply it by `0.621371` to convert to miles.

* **Program**:

  ```cpp

      return 0;
  }
  ```

### 13. Write a Program to Calculate the Perimeter of a Circle

* **Problem Definition**: Calculate the perimeter of a circle.
* **Problem Analysis**: Use the formula: `Perimeter = 2 * pi * radius`.
* **Algorithm**:

  1. Input the radius of the circle.
  2. Calculate the perimeter.
  3. Output the result.
* **Program**:

  ```cpp
  #include <iostream>
  #define PI 3.14159
  using namespace std;

  int main() {
      float radius, perimeter;
      cout << "Enter radius of the circle: ";
      cin >> radius;
      perimeter = 2 * PI * radius;
      cout << "Perimeter: " << perimeter << endl;
      return 0;
  }
  ```
