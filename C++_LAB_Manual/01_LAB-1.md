### 7. Flowchart to Find the Largest of Two Numbers

**Problem Definition:**
Develop an algorithm using pseudo-code and flowchart to find the larger of two numbers.

**Problem Analysis:**
- Requires comparison operation
- Decision-making structure (if-else)
- Two inputs, one output
- Demonstrates conditional logic

**Algorithm:**
1. Start
2. Read first number (num1)
3. Read second number (num2)
4. If num1 > num2, display num1
5. Else, display num2
6. End

**Pseudo-code:**
```
BEGIN
    INPUT num1
    INPUT num2
    IF num1 > num2 THEN
        OUTPUT num1
    ELSE
        OUTPUT num2
    ENDIF
END
```

**Flowchart Description:**
```
Start → Input num1 → Input num2 → 
Is num1 > num2? → Yes → Display num1 → End
                → No → Display num2 → End
```

### 13. Calculate the Perimeter of a Circle

**Problem Definition:**
Write a program to calculate the perimeter (circumference) of a circle given its radius.

**Problem Analysis:**
- Mathematical formula: P = 2 × π × r
- Constant value for π needed
- Floating-point data type recommended
- One input, one output

**Algorithm:**
1. Start
2. Read radius of circle
3. Calculate perimeter = 2 × π × radius
4. Display perimeter
5. End

**Program:**
```cpp
#include <iostream>
using namespace std;

int main() {
    const float PI = 3.14159;
    float radius, perimeter;
    
    cout << "Enter radius of the circle: ";
    cin >> radius;
    
    perimeter = 2 * PI * radius;
    cout << "Perimeter: " << perimeter << endl;
    
    return 0;
}
```

### 14. Pseudo-Code for Computing Average of Five Numbers

**Problem Definition:**
Develop an algorithm using pseudo-code to calculate the average of five numbers.

**Problem Analysis:**
- Requires five inputs
- Summation and division operations
- Demonstrates multiple input handling
- Basic statistical calculation

**Algorithm:**
1. Start
2. Read five numbers
3. Calculate total = sum of all five numbers
4. Calculate average = total / 5
5. Display average
6. End

**Pseudo-code:**
```
BEGIN
    INPUT num1, num2, num3, num4, num5
    total = num1 + num2 + num3 + num4 + num5
    average = total / 5
    OUTPUT average
END
```

**Program Implementation:**
```cpp
#include <iostream>
using namespace std;

int main() {
    float num1, num2, num3, num4, num5, average;
    
    cout << "Enter five numbers: ";
    cin >> num1 >> num2 >> num3 >> num4 >> num5;
    
    average = (num1 + num2 + num3 + num4 + num5) / 5;
    cout << "Average: " << average << endl;
    
    return 0;
}
```

```cpp
#include <iostream>
```

```cpp
    int a, b;

    cout << "Addition: " << a + b << endl;

    cout << "Multiplication: " << a * b << endl;
    
    if (b != 0) {
```
