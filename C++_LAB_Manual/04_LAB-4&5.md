### 1. Check if a Number is Even or Odd Using `if`

* **Problem Definition**: Determine if a number is even or odd.
* **Problem Analysis**: Use the modulus operator `%` to check if the number is divisible by 2.
* **Algorithm**:

  1. Input a number.
  2. Check if the number is divisible by 2.
  3. Print "Even" or "Odd".
* **Program**:

  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
      int number;
      cout << "Enter a number: ";
      cin >> number;

      if (number % 2 == 0)
          cout << "Even" << endl;
      else
          cout << "Odd" << endl;

      return 0;
  }
  ```

### 8. Display the Day of the Week Using `switch`

* **Problem Definition**: Display the day of the week using a `switch` statement.
* **Problem Analysis**: Use `switch` to display the day corresponding to a number (1 for Sunday, 7 for Saturday).
* **Algorithm**:

  1. Input a number from 1 to 7.
  2. Use `switch` to print the corresponding day.
  3. Output the result.
* **Program**:

  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
      int day;
      cout << "Enter a number (1-7): ";
      cin >> day;

      switch (day) {
          case 1: cout << "Sunday" << endl; break;
          case 2: cout << "Monday" << endl; break;
          case 3: cout << "Tuesday" << endl; break;
          case 4: cout << "Wednesday" << endl; break;
          case 5: cout << "Thursday" << endl; break;
          case 6: cout << "Friday" << endl; break;
          case 7: cout << "Saturday" << endl; break;
          default: cout << "Invalid input" << endl;
      }

      return 0;
  }
  ```

### 9. Simple Calculator Using `switch`

* **Problem Definition**: Implement a simple calculator using a `switch` statement.
* **Problem Analysis**


: Use `switch` to perform basic arithmetic operations (+, -, *, /).

* **Algorithm**:

  1. Input two numbers and an operator.
  2. Perform the operation based on the operator using `switch`.
  3. Output the result.
* **Program**:

  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
      int num1, num2;
      char op;

      cout << "Enter two numbers and an operator (+, -, *, /): ";
      cin >> num1 >> num2 >> op;

      switch (op) {
          case '+': cout << "Result: " << num1 + num2 << endl; break;
          case '-': cout << "Result: " << num1 - num2 << endl; break;
          case '*': cout << "Result: " << num1 * num2 << endl; break;
          case '/': 
              if (num2 != 0)
                  cout << "Result: " << num1 / num2 << endl;
              else
                  cout << "Cannot divide by zero" << endl;
              break;
          default: cout << "Invalid operator" << endl;
      }

      return 0;
  }
  ```

### 10. Check if a Year is a Leap Year Using Nested `if`

* **Problem Definition**: Check if a year is a leap year using nested `if`.
* **Problem Analysis**: A year is a leap year if it is divisible by 4, but not divisible by 100, except when divisible by 400.
* **Algorithm**:

  1. Input the year.
  2. Use nested `if` statements to check the leap year condition.
  3. Output the result.
* **Program**:

  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
      int year;
      cout << "Enter a year: ";
      cin >> year;

      if (year % 4 == 0) {
          if (year % 100 == 0) {
              if (year % 400 == 0)
                  cout << year << " is a leap year." << endl;
              else
                  cout << year << " is not a leap year." << endl;
          } else {
              cout << year << " is a leap year." << endl;
          }
      } else {
          cout << year << " is not a leap year." << endl;
      }

      return 0;
  }
  ```
