### 9. Simple Calculator Using `switch`

**Algorithm:**
1. Start
2. Declare integer variables num1, num2 and character variable op
3. Display "Enter two numbers and an operator (+, -, *, /): "
4. Read num1, num2, op from user input
5. Use switch statement on op:
   - Case '+': Display num1 + num2
   - Case '-': Display num1 - num2
   - Case '*': Display num1 * num2
   - Case '/': Check if num2 != 0, then display num1/num2, else error
   - Default: Display "Invalid operator"
6. End

**Program:**
```cpp
#include <iostream>
using namespace std;

int main() {
    int num1, num2;
    char op;
    
    cout << "Enter two numbers and an operator (+, -, *, /): ";
    cin >> num1 >> num2 >> op;
    
    switch (op) {
        case '+': 
            cout << "Result: " << num1 + num2 << endl;
            break;
        case '-': 
            cout << "Result: " << num1 - num2 << endl;
            break;
        case '*': 
            cout << "Result: " << num1 * num2 << endl;
            break;
        case '/': 
            if (num2 != 0)
                cout << "Result: " << num1 / num2 << endl;
            else
                cout << "Cannot divide by zero" << endl;
            break;
        default: 
            cout << "Invalid operator" << endl;
    }
    
    return 0;
}
```

### 12. Find Roots of Quadratic Equation

**Algorithm:**
1. Start
2. Declare float variables a, b, c, discriminant, root1, root2
3. Display "Enter coefficients a, b, c: "
4. Read a, b, c from user input
5. Calculate discriminant = b² - 4ac
6. If discriminant > 0: two real distinct roots
7. Else if discriminant == 0: two real equal roots
8. Else: complex roots
9. Display appropriate roots
10. End

**Program:**
```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    float a, b, c, discriminant, root1, root2;
    
    cout << "Enter coefficients a, b, c: ";
    cin >> a >> b >> c;
    
    discriminant = b * b - 4 * a * c;
    
    if (discriminant > 0) {
        root1 = (-b + sqrt(discriminant)) / (2 * a);
        root2 = (-b - sqrt(discriminant)) / (2 * a);
        cout << "Two distinct real roots: " << root1 << " and " << root2 << endl;
    } else if (discriminant == 0) {
        root1 = -b / (2 * a);
        cout << "Two equal real roots: " << root1 << endl;
    } else {
        cout << "Complex roots" << endl;
    }
    
    return 0;
}
```

### 13. Check if a Number is Prime

**Algorithm:**
1. Start
2. Declare integer variable number, flag = 0
3. Display "Enter a number: "
4. Read number from user input
5. If number <= 1, not prime
6. Else check divisibility from 2 to number/2
7. If divisible by any number, not prime
8. Else prime
9. Display result
10. End

**Program:**
```cpp
#include <iostream>
using namespace std;

int main() {
    int number;
    bool isPrime = true;
    
    cout << "Enter a number: ";
    cin >> number;
    
    if (number <= 1) {
        isPrime = false;
    } else {
        for (int i = 2; i <= number / 2; i++) {
            if (number % i == 0) {
                isPrime = false;
                break;
            }
        }
    }
    
    if (isPrime)
        cout << number << " is prime" << endl;
    else
        cout << number << " is not prime" << endl;
    
    return 0;
}
```

### 14. Calculate Electricity Bill

**Algorithm:**
1. Start
2. Declare float variable units, bill
3. Display "Enter electricity units consumed: "
4. Read units from user input
5. Calculate bill based on slab rates:
   - First 50 units: ₹0.50/unit
   - Next 100 units: ₹0.75/unit
   - Next 100 units: ₹1.20/unit
   - Above 250 units: ₹1.50/unit
6. Add 20% surcharge
7. Display total bill
8. End

**Program:**
```cpp
#include <iostream>
using namespace std;

int main() {
    float units, bill = 0;
    
    cout << "Enter electricity units consumed: ";
    cin >> units;
    
    if (units <= 50)
        bill = units * 0.50;
    else if (units <= 150)
        bill = 50 * 0.50 + (units - 50) * 0.75;
    else if (units <= 250)
        bill = 50 * 0.50 + 100 * 0.75 + (units - 150) * 1.20;
    else
        bill = 50 * 0.50 + 100 * 0.75 + 100 * 1.20 + (units - 250) * 1.50;
    
    bill += bill * 0.20; // 20% surcharge
    
    cout << "Total electricity bill: ₹" << bill << endl;
    
    return 0;
}
```

### 15. Menu-Driven Program Using `switch`

**Algorithm:**
1. Start
2. Declare integer variables choice, num1, num2, result
3. Display menu options:
   - 1. Add
   - 2. Subtract
   - 3. Multiply
   - 4. Divide
   - 5. Exit
4. Read choice from user input
5. If choice 1-4, read two numbers
6. Use switch to perform operation
7. Display result
8. If choice 5, exit
9. End

**Program:**
```cpp
#include <iostream>
using namespace std;

int main() {
    int choice, num1, num2, result;
    
    do {
        cout << "\nMenu:" << endl;
        cout << "1. Add" << endl;
        cout << "2. Subtract" << endl;
        cout << "3. Multiply" << endl;
        cout << "4. Divide" << endl;
        cout << "5. Exit" << endl;
        cout << "Enter your choice: ";
        cin >> choice;
        
        if (choice >= 1 && choice <= 4) {
            cout << "Enter two numbers: ";
            cin >> num1 >> num2;
        }
        
        switch (choice) {
            case 1:
                result = num1 + num2;
                cout << "Result: " << result << endl;
                break;
            case 2:
                result = num1 - num2;
                cout << "Result: " << result << endl;
                break;
            case 3:
                result = num1 * num2;
                cout << "Result: " << result << endl;
                break;
            case 4:
                if (num2 != 0) {
                    result = num1 / num2;
                    cout << "Result: " << result << endl;
                } else {
                    cout << "Cannot divide by zero" << endl;
                }
                break;
            case 5:
                cout << "Exiting program..." << endl;
                break;
            default:
                cout << "Invalid choice" << endl;
        }
    } while (choice != 5);
    
    return 0;
}
```
