### 13. Quadratic Equation Solver

**Algorithm:**
1. Start
2. Declare float variables a, b, c, discriminant, root1, root2
3. Display "Enter values for a, b, and c: "
4. Read a, b, c from user input
5. Calculate discriminant = b² - 4ac
6. If discriminant > 0:
   - Calculate two real roots
   - Display both roots
7. Else if discriminant == 0:
   - Calculate single real root
   - Display the root
8. Else display "complex roots"
9. End

**Program:**
```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    float a, b, c, discriminant, root1, root2;
    
    cout << "Enter values for a, b, and c: ";
    cin >> a >> b >> c;
    
    discriminant = b * b - 4 * a * c;
    
    if (discriminant > 0) {
        root1 = (-b + sqrt(discriminant)) / (2 * a);
        root2 = (-b - sqrt(discriminant)) / (2 * a);
        cout << "Roots are real and different." << endl;
        cout << "Root 1 = " << root1 << endl;
        cout << "Root 2 = " << root2 << endl;
    } else if (discriminant == 0) {
        root1 = -b / (2 * a);
        cout << "Roots are real and same." << endl;
        cout << "Root 1 = Root 2 = " << root1 << endl;
    } else {
        cout << "Roots are complex." << endl;
    }
    
    return 0;
}
```

### 14. Total and Percentage of 5 Subjects

**Algorithm:**
1. Start
2. Declare float array marks[5], total = 0, percentage
3. Display "Enter marks for 5 subjects: "
4. For i = 0 to 4:
   - Read marks[i]
   - Add marks[i] to total
5. Calculate percentage = total ÷ 5
6. Display total and percentage
7. End

**Program:**
```cpp
#include <iostream>
using namespace std;

int main() {
    float marks[5], total = 0, percentage;
    
    cout << "Enter marks for 5 subjects: ";
    for (int i = 0; i < 5; i++) {
        cin >> marks[i];
        total += marks[i];
    }
    
    percentage = total / 5;
    cout << "Total marks: " << total << endl;
    cout << "Percentage: " << percentage << "%" << endl;
    
    return 0;
}
```

### 15. Largest of Three Numbers Using Ternary Operator

**Algorithm:**
1. Start
2. Declare integer variables num1, num2, num3, largest
3. Display "Enter three numbers: "
4. Read num1, num2, num3 from user input
5. Find largest using nested ternary operator
6. Display largest number
7. End

**Program:**
```cpp
#include <iostream>
using namespace std;

int main() {
    int num1, num2, num3, largest;
    
    cout << "Enter three numbers: ";
    cin >> num1 >> num2 >> num3;
    
    largest = (num1 >= num2 && num1 >= num3) ? num1 : 
              (num2 >= num1 && num2 >= num3) ? num2 : num3;
    
    cout << "The largest number is: " << largest << endl;
    
    return 0;
}
```
