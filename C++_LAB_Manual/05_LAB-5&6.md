* **Problem Analysis**: Use a `for` loop to iterate from 1 to 10 and print each number.
* **Algorithm**:

### 5. Compute Factorial Using `while`

* **Problem Definition**: Compute the factorial of a given number using a `while` loop.
* **Problem Analysis**: Use a `while` loop to multiply all numbers from 1 to the given number.
* **Algorithm**:

  1. Input a number `n`.
  2. Initialize `factorial` to 1.
  3. Use a `while` loop to multiply `factorial` by all numbers from 1 to `n`.
  4. Output the result.
* **Program**:

  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
      int num;
      long long factorial = 1;
      cout << "Enter a number: ";
      cin >> num;

      int i = 1;
      while (i <= num) {
          factorial *= i;
          i++;
      }

      cout << "Factorial of " << num << " is: " << factorial << endl;
      return 0;
  }
  ```

### 6. Print Fibonacci Sequence Up to N Terms

* **Problem Definition**: Print the Fibonacci sequence up to `N` terms.
* **Problem Analysis**: Use a `for` loop to generate the Fibonacci numbers. The next term is the sum of the previous two terms.
* **Algorithm**:

  1. Input the number of terms `N`.
  2. Initialize the first two terms of the Fibonacci sequence.
  3. Use a loop to generate the sequence up to `N` terms.
  4. Output the sequence.
* **Program**:

  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
      int n;
      cout << "Enter the number of terms: ";
      cin >> n;

      int t1 = 0, t2 = 1, nextTerm;
      cout << "Fibonacci Sequence: ";
      
      for (int i = 1; i <= n; i++) {
          cout << t1 << " ";
          nextTerm = t1 + t2;
          t1 = t2;
          t2 = nextTerm;
      }
      cout << endl;
      return 0;
  }
  ```

### 7. Reverse Digits of an Integer

* **Problem Definition**: Reverse the digits of a given integer.
* **Problem Analysis**: Use a `while` loop to extract each digit and construct the reversed number.
* **Algorithm**:

  1. Input the number.
  2. Use a `while` loop to extract digits and build the reversed number.
  3. Output the reversed number.
* **Program**:

  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
      int num, reversed = 0;
      cout << "Enter a number: ";
      cin >> num;

      while (num != 0) {
          int digit = num % 10;
          reversed = reversed * 10 + digit;
          num /= 10;
      }

      cout << "Reversed number: " << reversed << endl;
      return 0;
  }
  ```

### 10. Check Whether a Number is a Palindrome

* **Problem Definition**: Check if a given number is a palindrome.
* **Problem Analysis**:


A number is a palindrome if it reads the same backward as forward.

* **Algorithm**:

  1. Input the number.
  2. Reverse the digits of the number.
  3. Compare the reversed number with the original.
  4. Output whether the number is a palindrome or not.
* **Program**:

  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
      int num, original, reversed = 0;
      cout << "Enter a number: ";
      cin >> num;

      original = num;
      while (num != 0) {
          reversed = reversed * 10 + num % 10;
          num /= 10;
      }

      if (original == reversed)
          cout << original << " is a palindrome." << endl;
      else
          cout << original << " is not a palindrome." << endl;

      return 0;
  }
  ```

### 11. Check if a Number is Prime

* **Problem Definition**: Check if a number is prime.
* **Problem Analysis**: A prime number is only divisible by 1 and itself.
* **Algorithm**:

  1. Input the number.
  2. Check if the number is divisible by any number from 2 to √n.
  3. If it is divisible, it's not prime; otherwise, it's prime.
* **Program**:

  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
      int num;
      bool isPrime = true;

      cout << "Enter a number: ";
      cin >> num;

      if (num <= 1) {
          isPrime = false;
      } else {
          for (int i = 2; i * i <= num; i++) {
              if (num % i == 0) {
                  isPrime = false;
                  break;
              }
          }
      }

      if (isPrime)
          cout << num << " is a prime number." << endl;
      else
          cout << num << " is not a prime number." << endl;

      return 0;
  }
  ```

### 12. Print Prime Numbers Between 1 and 100

* **Problem Definition**: Print all the prime numbers between 1 and 100.
* **Problem Analysis**: A prime number is a number greater than 1 that has no divisors other than 1 and itself. We can check if a number is prime by attempting to divide it by all numbers from 2 to the square root of the number.
* **Algorithm**:

  1. Loop through all numbers from 2 to 100.
  2. For each number, check if it's divisible by any number other than 1 and itself.
  3. If it's not divisible, it is prime and should be printed.
* **Program**:

  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
      for (int num = 2; num <= 100; num++) {
          bool isPrime = true;
          
          for (int i = 2; i * i <= num; i++) {
              if (num % i == 0) {
                  isPrime = false;
                  break;
              }
          }
          
          if (isPrime) {
              cout << num << " ";
          }
      }
      cout << endl;
      return 0;
  }
  ```

### 13. Generate Pattern of Stars (Pyramid)

* **Problem Definition**: Generate a pyramid of stars (`*`) with a given number of rows.

* **Problem Analysis**: We need to print a pattern of stars where the number of stars in each row increases as we go down, with spaces on the left to center the stars.

* **Algorithm**:

  1. Input the number of rows.
  2. For each row, print spaces followed by stars.
  3. The number of spaces decreases while the number of stars increases with each row.

* **Program**:

  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
      int rows;
      cout << "Enter number of rows: ";
      cin >> rows;

      for (int i = 1; i <= rows; i++) {
          // Print spaces
          for (int j = i; j < rows; j++) {
              cout << " ";
          }

          // Print stars
          for (int j = 1; j <= (2 * i - 1); j++) {
              cout << "*";
          }

          cout << endl;
      }
      return 0;
  }
  ```

  **Example Output** for `rows = 5`:

  ```
      *
     ***
    *****
   *******
  *********
  ```

### 15. Calculate Power of a Number Using Loops

* **Problem Definition**: Calculate the power of a number (base^exponent) using loops.

* **Problem Analysis**: We can calculate the power of a number by multiplying the base by itself repeatedly for the number of times equal to the exponent.

* **Algorithm**:

  1. Input the base and exponent.
  2. Initialize a variable `result` to 1.
  3. Use a loop to multiply the base by `result` exponent times.
  4. Output the result.

* **Program**:

  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
      int base, exponent;
      long long result = 1;

      cout << "Enter base and exponent: ";
      cin >> base >> exponent;

      for (int i = 1; i <= exponent; i++) {
          result *= base;
      }

      cout << base << " raised to the power of " << exponent << " is: " << result << endl;
      return 0;
  }
  ```

  **Example Output** for `base = 2` and `exponent = 3`:

  ```
  2 raised to the power of 3 is: 8
  ```
