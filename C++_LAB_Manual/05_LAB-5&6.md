### 1. Print Numbers from 1 to 10 Using `for`

**Algorithm:**
1. Start
2. Initialize integer variable i = 1
3. For i = 1 to 10:
   - Display i
   - Increment i by 1
4. End

**Program:**
```cpp
#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 10; i++) {
        cout << i << " ";
    }
    cout << endl;
    return 0;
}
```

3. Initialize factorial = 1, i = 1

### 6. Print Fibonacci Sequence Up to N Terms

**Algorithm:**
1. Start
2. Read integer n from user
3. Initialize t1 = 0, t2 = 1
4. Display "Fibonacci Sequence: "
5. For i = 1 to n:
   - Display t1
   - Calculate nextTerm = t1 + t2
   - Update t1 = t2, t2 = nextTerm
6. End

**Program:**
```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cout << "Enter the number of terms: ";
    cin >> n;

    int t1 = 0, t2 = 1, nextTerm;
    cout << "Fibonacci Sequence: ";
    
    for (int i = 1; i <= n; i++) {
        cout << t1 << " ";
        nextTerm = t1 + t2;
        t1 = t2;
        t2 = nextTerm;
    }
    cout << endl;
    return 0;
}
```

### 7. Reverse Digits of an Integer

**Algorithm:**
1. Start
2. Read integer num from user
3. Initialize reversed = 0
4. While num != 0:
   - Extract last digit: digit = num % 10
   - Build reversed number: reversed = reversed × 10 + digit
   - Remove last digit: num = num / 10
5. Display reversed
6. End

**Program:**
```cpp
#include <iostream>
using namespace std;

int main() {
    int num, reversed = 0;
    cout << "Enter a number: ";
    cin >> num;

    while (num != 0) {
        int digit = num % 10;
        reversed = reversed * 10 + digit;
        num /= 10;
    }

    cout << "Reversed number: " << reversed << endl;
    return 0;
}
```

### 10. Check Whether a Number is a Palindrome

**Algorithm:**
1. Start
2. Read integer num from user, store original = num
3. Initialize reversed = 0
4. While num != 0:
   - Extract digit and build reversed number
5. If original == reversed, display "palindrome"
6. Else display "not palindrome"
7. End

**Program:**
```cpp
#include <iostream>
using namespace std;

int main() {
    int num, original, reversed = 0;
    cout << "Enter a number: ";
    cin >> num;

    original = num;
    while (num != 0) {
        reversed = reversed * 10 + num % 10;
        num /= 10;
    }

    if (original == reversed)
        cout << original << " is a palindrome." << endl;
    else
        cout << original << " is not a palindrome." << endl;

    return 0;
}
```

### 11. Check if a Number is Prime

**Algorithm:**
1. Start
2. Read integer num from user
3. Initialize isPrime = true
4. If num <= 1, set isPrime = false
5. Else for i = 2 to √num:
   - If num divisible by i, set isPrime = false, break
6. Display result
7. End

**Program:**
```cpp
#include <iostream>
using namespace std;

int main() {
    int num;
    bool isPrime = true;

    cout << "Enter a number: ";
    cin >> num;

    if (num <= 1) {
        isPrime = false;
    } else {
        for (int i = 2; i * i <= num; i++) {
            if (num % i == 0) {
                isPrime = false;
                break;
            }
        }
    }

    if (isPrime)
        cout << num << " is a prime number." << endl;
    else
        cout << num << " is not a prime number." << endl;

    return 0;
}
```

### 13. Generate Pattern of Stars (Pyramid)

**Algorithm:**
1. Start
2. Read integer rows from user
3. For i = 1 to rows:
   - Print (rows - i) spaces
   - Print (2*i - 1) stars
   - Move to next line
4. End

**Program:**
```cpp
#include <iostream>
using namespace std;

int main() {
    int rows;
    cout << "Enter number of rows: ";
    cin >> rows;

    for (int i = 1; i <= rows; i++) {
        // Print spaces
        for (int j = i; j < rows; j++) {
            cout << " ";
        }

        // Print stars
        for (int j = 1; j <= (2 * i - 1); j++) {
            cout << "*";
        }

        cout << endl;
    }
    return 0;
}
```

**Example Output for rows = 5:**
```
    *
   ***
  *****
 *******
*********
```

### 15. Calculate Power of a Number Using Loops

**Algorithm:**
1. Start
2. Read base and exponent from user
3. Initialize result = 1
4. For i = 1 to exponent:
   - Multiply result by base
5. Display result
6. End

**Program:**
```cpp
#include <iostream>
using namespace std;

int main() {
    int base, exponent;
    long long result = 1;

    cout << "Enter base and exponent: ";
    cin >> base >> exponent;

    for (int i = 1; i <= exponent; i++) {
        result *= base;

    return 0;
}
```
